MiniKube Commands
===================
minikube start
minikube status -p cluster2
minikube start --nodes 2 -p multinode-demo. #Start one master and one worker
minikube stop -p cluster2

kubectl config view
**With  configuration file
export KUBECONFIG=multidemo_config
